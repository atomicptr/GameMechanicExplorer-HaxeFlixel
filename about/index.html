About

This site is a HaxeFlixel port of John Watsons: Game Mechanic Explorer which was written for the JavaScript game framework Phaser.

Purpose?

The purpose of this site is obviously to push the awesome HaxeFlixel framework ;).

How to contribute?

If you want to contribute more awesome tutorials you can press the "Github" button at the top and fork the project!
