Game Mechanic Explorer

A collection of concrete examples for various game mechanics, algorithms, and effects. The examples are all implemented in Haxe using the HaxeFlixel game framework, but the concepts and methods are general and can be adapted to any engine. Think of it as pseudocode. Each section contains several different examples that progress in sequence from a very basic implementation to a more advanced implementation.
